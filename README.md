## 🛠️ ၁။ အသုံးပြုရန် လိုအပ်သော ပရိုတိုကောများ (Required Protocols)

ဒရုန်းမှ ဗီဒီယိုကို မြေပြင်သို့ ၈ ကီလိုမီတာအထိ ပို့လွှတ်ရန် အောက်ပါ Protocol များကို ပေါင်းစပ်အသုံးပြုရပါမည်-
* **Video Encoding Protocol (H.264 / H.265):** ဗီဒီယိုဖိုင် အရွယ်အစားကို သေးငယ်သွားစေရန် (Compress) လုပ်ပေးသော စနစ်။ H.265 (HEVC) ကို အသုံးပြုခြင်းက အကောင်းဆုံးဖြစ်ပါသည်။
* **Transmission Protocol (COFDM):** အဆောက်အဦးများ၊ သစ်ပင်များနှင့် လေထုအတွင်း လှိုင်းများရိုက်ခတ်ခြင်းကို အကောင်းဆုံး ကျော်လွှားနိုင်တဲ့ ဒစ်ဂျစ်တယ် လှိုင်းထုတ်လွှင့်မှု စနစ်ဖြစ်ပါသည်။
* **Wifibroadcast Protocol (For Wi-Fi 7):** Standard Wi-Fi Protocol မှ ACK စနစ်ကို ဖယ်ရှားပြီး UDP Packet များကို Monitor Mode မှတဆင့် လေထဲသို့ တိုက်ရိုက်ပက်ဖြန်း (Broadcast) ပေးသော နည်းပညာဖြစ်ပါသည်။ (OpenHD / Ruby FPV မှ အသုံးပြုသည်)
* **FEC (Forward Error Correction):** လှိုင်းထုတ်လွှင့်စဉ် ဒေတာအချို့ ပျောက်ဆုံးသွားပါက မြေပြင်စခန်းမှ ဒေတာကို ပြန်လည်တွက်ချက် တည်ဆောက်ပေးသည့် စနစ်ဖြစ်သည်။

## 🚀 ၂။ Wi-Fi 7 (802.11be) ဖြင့် ၈ ကီလိုမီတာ ထုတ်လွှင့်ခြင်း
